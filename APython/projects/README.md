# 🐍 Advanced Python Projects — APython

A collection of real-world Python applications covering billing, security, management systems, and simulations. These projects demonstrate practical use of Python in building functional, production-style tools.

---

## 📁 Projects

| File | Description | Key Concept |
|------|-------------|-------------|
| `billing_system_for_retail.py` | A retail billing system that generates itemized bills with totals and taxes | OOP, arithmetic, file I/O |
| `cart.py` | A shopping cart system to add, remove, and manage items before checkout | Lists, dictionaries, OOP |
| `college_result_management.py` | Manages student results — store, update, and display academic records | File handling, data management |
| `freelancer_marketing_samulation.py` | Simulates a freelancer's marketing workflow — clients, projects, and earnings | OOP, simulation logic |
| `movie_ticket_bookingsystem.py` | A movie ticket booking system with seat selection and pricing | OOP, conditionals, user input |
| `parking_lot_management.py` | Manages parking slot allocation, entry/exit tracking, and fee calculation | OOP, time logic |
| `passward_manager.py` | A password manager to store and retrieve credentials securely | File I/O, encryption basics |
| `password_checker.py` | Checks password strength based on length, characters, and complexity rules | String methods, conditionals |
| `to_do_list_manager.py` | A to-do list app to add, complete, and delete tasks | File persistence, lists |
| `transaction_system.py` | Simulates financial transactions with balance tracking and history | OOP, arithmetic, data handling |

### 🔐 Supporting Files

| File | Description |
|------|-------------|
| `key.key` | Encryption key used by the password manager for securing stored credentials |
| `passwords.enc` | Encrypted file storing saved passwords (generated by `passward_manager.py`) |

---

## 🧠 Concepts Covered

- Object-Oriented Programming (classes, methods, inheritance)
- File handling — reading, writing, and encrypted storage
- User input and input validation
- Data management — CRUD operations on records
- Simulation logic and workflow modeling
- Security basics — password encryption and strength checking
- Financial and business logic (billing, transactions, parking fees)

## ▶️ How to Run

**Prerequisites:** Python 3.x

```bash
# Clone the repository
git clone https://github.com/siddharth9508kumar/advance-python-projects-.git

# Navigate to the projects folder
cd "advance-python-projects-/APython/projects"

# Run any project
python billing_system_for_retail.py
python movie_ticket_bookingsystem.py
python to_do_list_manager.py
```

> **Note:** `passward_manager.py` requires the `cryptography` library.
> Install it with: `pip install cryptography`
