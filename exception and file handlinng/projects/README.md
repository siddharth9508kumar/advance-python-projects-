# 🐍 Advanced Python Projects — Exception & File Handling

A collection of advanced Python projects demonstrating real-world exception handling, custom error frameworks, and file management patterns.

## 📁 Projects

| Project | Description |
|--------|-------------|
| **Banking System with Transactions** | Simulates a banking system with transaction management, handling errors like insufficient funds, invalid accounts, and failed transfers. |
| **Contact Book** | A contact management application with file-based persistence and robust exception handling for duplicate entries, missing contacts, and I/O errors. |
| **Custom Exception Framework** | A reusable exception hierarchy and error-handling framework demonstrating how to design and raise custom exceptions in Python. |
| **E-Commerce Order Management** | An order processing system that handles inventory errors, payment failures, and invalid order states using structured exception handling. |
| **Flight Booking System** | A flight reservation system that manages booking errors, seat availability conflicts, and invalid route/date inputs. |
| **Student Grade Management System** | Manages student records and grades with validation logic and error handling for invalid inputs, missing students, and file I/O operations. |

## 🧠 Concepts Covered

- `try` / `except` / `else` / `finally` blocks
- Custom exception classes and inheritance hierarchies
- File reading and writing with proper error handling
- Exception chaining and re-raising
- Input validation and defensive programming
- Real-world application design patterns

## 🚀 Getting Started

**Prerequisites:** Python 3.8+

```bash
# Clone the repository
git clone https://github.com/siddharth9508kumar/advance-python-projects-.git

# Navigate to the projects folder
cd "advance-python-projects-/exception and file handlinng/projects"

# Run any project
python "Banking System with Transactions.py"
```
