# 🐍 Advance Python Projects

A comprehensive collection of Python projects and programs organized by topic — covering core functions, file & exception handling, and advanced real-world applications.

---

## 📁 Repository Structure

```
advance-python-projects-/
├── APython/                        # Real-world application projects
├── exception and file handlinng/   # File I/O and exception handling programs
└── functions/                      # Core Python function and logic programs
```

## 🗂️ Folders

### 🔷 [APython](./APython)
Real-world Python applications simulating practical systems like billing, booking, password management, and financial transactions.

**Highlights:**
- Retail Billing System
- Movie Ticket Booking System
- Password Manager (with encryption)
- Parking Lot Management
- Shopping Cart
- To-Do List Manager
- Transaction System
- College Result Management
- Freelancer Marketing Simulation
- Password Strength Checker

---

### 🔷 [Exception and File Handling](./exception%20and%20file%20handlinng)
Programs from PPML Lab Experiment 9 demonstrating file operations and exception handling in Python.

**Highlights:**
- Create, read, and write text files
- Count lines, words, characters, and vowels
- Handle ZeroDivisionError, ValueError, FileNotFoundError
- Use of `try`, `except`, `else`, and `finally` blocks

## ▶️ Getting Started

**Prerequisites:** Python 3.x

```bash
# Clone the repository
git clone https://github.com/siddharth9508kumar/advance-python-projects-.git

# Navigate into any folder
cd "advance-python-projects-/APython/projects"
cd "advance-python-projects-/functions"
cd "advance-python-projects-/exception and file handlinng/projects"

# Run any Python file
python <filename>.py
```

> Some projects in `APython` may require third-party libraries.
> Install dependencies with: `pip install cryptography`
